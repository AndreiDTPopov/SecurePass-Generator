## Reliable Password Generator üîëüöÄ

A simple Python application designed to create strong and reliable passwords of varying lengths and complexities.  

## Installation & Usage
1. Ensure you have Python 3.x installed on your system.  
2. Download or clone the project from GitHub.  
3. Open a terminal window and navigate to the project directory.  
4. Run the following command: python password_generator.py

Follow the prompts in the terminal, enter the desired password length, and select options for including special symbols, numbers, and uppercase letters.  

## Features
Flexible configuration of password parameters.  
Support for major character classes (digits, lowercase and uppercase letters, special symbols).  
User-friendly command-line interface.
